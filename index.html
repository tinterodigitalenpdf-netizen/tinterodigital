Supabase no configurado — usando almacenamiento local
Conectá Supabase para sincronizar el catálogo entre todos tus dispositivos. Es gratis y no necesita instalar nada.

☁ Cómo configurar Supabase (5 min)

1. Entrá a supabase.com → New project → elegí nombre y región.
2. En el editor SQL (SQL Editor → New query), ejecutá esto:

create table books ( id text primary key, title text not null, authors text, thumb text, categories text, price numeric not null, lemon_url text not null, created_at timestamptz default now() ); alter table books enable row level security; create policy "lectura publica" on books for select using (true); create policy "admin total" on books for all using (true);
3. Ir a Project Settings → API → copiá la Project URL y la anon public key.
4. Pegá ambas en los campos de arriba y guardá. ¡Listo!

🚀 Cómo publicar

1. GitHub: Nuevo repositorio público → subí este index.html.
2. Cloudflare Pages: pages.cloudflare.com → Conectar GitHub → elegí el repo → deploy automático.
3. Dominio propio: Desde Cloudflare podés conectar tu dominio sin costo extra.

Sin Supabase: el catálogo vive en este navegador (localStorage). Con Supabase: se sincroniza en todos los dispositivos al instante.
